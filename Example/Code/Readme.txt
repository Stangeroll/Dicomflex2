To use the T1Mapper example, please:

-	copy the cfg_application_T1Mapper.json file in the same folder as the cControl.m file and make the cComputeT1Mapper.m available for the interpreter.
-	type "cControl" in the Matlab command window
-	a list dialog should now appear with "T1Mapper" as the only choice. -> choose it!
-	a load dialog will appear. Please select the "T1_PhantomData" folder
-	a question dialog will appear. You can freely choose to start a new session or use the existing one the program found in the folder.
